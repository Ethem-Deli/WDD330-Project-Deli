About This Project

The World History Timeline is an interactive web application designed to help learners and history enthusiasts explore significant world events through a visual, engaging interface.

🌍 Our Mission

Our mission is to make learning history accessible, interactive, and enjoyable. The timeline displays both local JSON events and real-time events fetched from an external API, combining educational content with live data.

💡 Features

  • 📅 Explore major historical milestones across centuries
  • ⭐ Save your favorite events for quick access
  • 🗺️ Interactive maps powered by Google Maps API
  • 📤 Export and share your favorite timelines
  • 📱 Fully responsive design with smooth animations

🧠 Technologies Used

This project is built with modern web technologies to ensure fast performance and maintainability:

  • HTML, CSS, JavaScript (ES Modules)
  • Vite and Github Pages for fast build and development
  • Google Maps JavaScript API
  • JSON-based data storage
  • GitHub Pages for hosting
